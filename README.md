# Amy's Lightweight Markup language (ALM)

Alm is a lightweight markup language similar to markdown.  I designed it for personal use, and it should not be considered stable until version 1.0.0.

This crate contains a library for parsing alm in your own rust programs, as well as a binary for command line processing.

In general, the markdown family of languages are way better for most use-cases and often have more complete and useful feature sets.  As such, markdown should be used instead of alm for most purposes.  Even this document is written in markdown.
